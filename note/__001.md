# 環境構築

CakePHP 5 + MySQL

## port

- 被らなそうなものを設定しておく
  - 28080:80
  - 23306:3306

## CakePHP アプリを作る

```bash
mkdir app
docker compose build
# Building 143.0s (13/13) FINISHED

docker compose run --rm app bash -lc "cd /var/www/html && composer create-project --prefer-dist cakephp/app:^5.2 ."
# Set Folder Permissions ? (Default to Y) [Y,n]? Y
```

